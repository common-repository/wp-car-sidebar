== Description ==

WP-Car adds descriptions and photos from famous cars in the sidebar of your blog. A configurable widget.
Support: [Autokredit](http://www.autokredit.de/). 

== Installation ==

1. Extract `wp-car-sidebar.1.0.zip`.
2. Upload the `wp-car-sidebar`-folder to your `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add the widget to your sidebar from Appearance->Widgets.
